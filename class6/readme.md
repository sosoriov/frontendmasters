# Storing your data

1. _mongodb_ -> [mlab](mlab.com)
    Free storage up to 500MB

    - Client: *Robomongo*

    examples using python and nodejs are available.
    
2. [_Firebase_](https://firebase.google.com/)
    Free storage up to 1GB
